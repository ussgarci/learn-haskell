Projects
--------

### lox
A Haskell implementation of the `lox` language from [Crafting Interpreters](https://craftinginterpreters.com/)


#### Building
```
cabal build
cabal run lox-exe <absolute path to lox file>
```
#### Testing
```
cabal build lox-test
cabal run lox-test
```
